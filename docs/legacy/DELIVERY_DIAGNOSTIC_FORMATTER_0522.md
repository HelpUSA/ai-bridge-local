# Delivery diagnostic formatter 0.5.22

Objetivo: padronizar a saida textual de diagnosticos readonly.

## Formato

- tipo=
- confianca=
- resumo=
- correcao=
- command_id=
- target_chat_id=
- status=
- readonly=

## Escopo

O formatter nao envia mensagens.
O formatter nao altera fila.
O formatter nao executa entrega inter-chat.
